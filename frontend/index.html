Project Snapshot

Healthy, Cavity, Misaligned

The project is structured to classify tooth conditions from images, compare KNN / Decision Tree / Naive Bayes, and provide a clear upgrade path into reconstruction.

Preprocess Resize + grayscale + flatten
Evaluate Accuracy, Precision, Recall, F1

Problem Statement

Classify tooth conditions from oral images.

The core objective is to label images as Healthy, Cavity, or Misaligned using lightweight machine learning methods.

Dataset

Image folders per class, processed into feature vectors.

The pipeline assumes class-organized raw images and produces a processed tabular dataset for classical ML.

Extension

3D reconstruction remains available as a higher-level module.

When needed, the repository also supports reconstruction workflows through the existing TeethDreamer stack.

Models Used

Baseline classifiers with an interpretable comparison view

KNN

k = 3

Simple neighborhood-based baseline

Useful as a sanity check when the dataset is small and class boundaries are locally structured.

  • Fast to train
  • Easy to explain
  • Good baseline for image features

Decision Tree

Interpretable

Rule-based model for decision tracing

Supports human-readable comparisons and feature-importance visualizations in the results folder.

  • Simple decision path
  • Feature importance available
  • Useful for presentations

Naive Bayes

Probabilistic

Compact probabilistic classifier

Provides a lightweight statistical baseline that can be compared directly against the other models.

  • Low computational cost
  • Strong baseline behavior
  • Useful on simple feature spaces

Ethical Considerations

Practical notes for research use

Privacy

Remove patient identifiers before storage or training and keep access limited to the project scope.

Fairness

Check whether model performance changes across classes or demographics and report those gaps clearly.

Transparency

Document the dataset source, preprocessing assumptions, and known limitations in the README and notebooks.
